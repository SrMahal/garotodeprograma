Error não link ao projeto

O erro ocorre porque a branch main do seu repositório local não está vinculada a um repositório remoto. Para corrigir isso, siga os passos abaixo:

1. Confirme que você tem um repositório remoto vinculado

Digite o seguinte comando no terminal:

git remote -v

Se não aparecer nada, você precisa adicionar o repositório remoto com:

git remote add origin "/URL_DO_SEU_REPOSITORIO"

2. Configure o upstream e envie os arquivos para o repositório

Como o Git sugere na mensagem de erro, você pode rodar o comando:

git push --set-upstream origin main

Isso define o repositório remoto e a branch main como padrão para futuros git push.

3. Caso já tenha o remoto configurado

Se o repositório já estiver vinculado (o comando git remote -v mostrar um repositório válido), tente:

git push -u origin main

Depois disso, os próximos git push podem ser feitos normalmente sem precisar definir o upstream novamente.
