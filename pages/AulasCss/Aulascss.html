Aulas Css

O Css ele serve para estilizar o HTML, ele serve para deixar o site mais bonito e visualmente agradavel

GRID

Criar um layout de grade em CSS é uma maneira poderosa de estruturar sua página da web. O CSS Grid Layout é um sistema de grade bidimensional que permite dividir uma página em linhas e colunas, facilitando a criação de layouts complexos

FLexBox

Flexbox é outro modelo de layout poderoso em CSS, especialmente útil para criar layouts unidimensionais, seja em linhas ou colunas. Ele fornece uma maneira eficiente de dispor, alinhar e distribuir espaço entre itens em um contêiner, mesmo quando seu tamanho é desconhecido ou dinâmico.

Animation
